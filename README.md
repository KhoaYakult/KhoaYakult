- Conducted extensive robustness testing against word and character-level noise.
- Deployed the model via **Gradio** web interface, optimizing the prediction logic to prioritize top $k$ highest probability outcomes rather than relying on fixed confidence thresholds.

- Built a visualization pipeline for analyzing renal health (kidneys, tumors, and cysts) from CT scans using the KiTS23 dataset.
- Implemented an **Enhanced Marching Cubes** algorithm, integrating Taubin smoothing and mesh decimation to generate highly accurate and optimized 3D models.

#### 3. Research RAG System for Academic Papers
- Developed a Retrieval-Augmented Generation (RAG) system specialized in querying and analyzing complex academic papers (e.g., CortexODE).
- Integrated **ChromaDB** for vector storage, **Ollama** for local embeddings, and **Qwen** models to ensure precise, low-latency information retrieval.

#### 4. Automated Quantitative Trading Foundation

#### 5. Data Processing & Feature Engineering Optimization
